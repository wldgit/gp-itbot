# Очистка и оптимизация рабочего компьютера

## 1. Назначение документа

Документ определяет правила поддержания работоспособности и производительности рабочих компьютеров сотрудников.

## 2. Область применения

Распространяется на все рабочие станции сотрудников.

## 3. Общие принципы

- Устройство должно работать стабильно и быстро
- Необходимо регулярно проводить базовое обслуживание
- Не допускать накопления лишних файлов и программ

## 4. Очистка диска

Рекомендуется:

- удалять временные файлы
- очищать корзину
- удалять ненужные загрузки
- проверять папки Downloads и Desktop

Периодичность:

- не реже 1 раза в месяц

## 5. Управление программами

- Удалять неиспользуемые программы
- Не устанавливать ПО без необходимости
- Использовать только разрешённые приложения

## 6. Автозагрузка

- Проверять список автозагрузки
- Отключать ненужные программы
- Оставлять только критичные сервисы

## 7. Обновления

- Устанавливать обновления ОС и ПО
- Перезагружать устройство при необходимости
- Не откладывать обновления без причины

## 8. Производительность

При снижении производительности:

- проверить загрузку CPU и RAM
- закрыть лишние приложения
- перезагрузить устройство
- обратиться в IT при необходимости

## 9. Хранение данных

- Не хранить большие объёмы данных локально
- Использовать корпоративные хранилища
- Удалять устаревшие файлы

## 10. Безопасность

- Не отключать защитные механизмы
- Проверять систему на вредоносное ПО при подозрении
- Не скачивать файлы из ненадёжных источников

## 11. Запрещённые действия

Запрещается:

- устанавливать неразрешённое ПО
- отключать системные службы
- использовать «чистящие» утилиты без согласования
- изменять системные настройки без необходимости

## 12. Ответственность

Несоблюдение правил может привести к снижению производительности, сбоям и угрозам безопасности.
