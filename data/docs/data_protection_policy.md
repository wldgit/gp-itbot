# Обработка и защита персональных данных

## 1. Назначение документа

Документ определяет правила обработки, хранения и защиты персональных данных в компании.

## 2. Область применения

Применяется ко всем сотрудникам, работающим с персональными данными клиентов, сотрудников и партнёров.

## 3. Определения

Персональные данные — любая информация, относящаяся к идентифицируемому физическому лицу, включая:

- ФИО
- email
- телефон
- адрес
- документы
- любые идентификаторы

## 4. Основные принципы обработки данных

- Законность и прозрачность
- Ограничение целей обработки
- Минимизация данных
- Точность и актуальность
- Ограничение хранения
- Обеспечение безопасности

## 5. Сбор данных

Разрешается:

- собирать только необходимые данные
- использовать утверждённые формы и системы

Запрещается:

- собирать избыточные данные
- использовать данные без понятной цели

## 6. Хранение данных

- Данные должны храниться только в утверждённых системах
- Доступ должен быть ограничен
- Данные должны быть защищены от утечек
- Срок хранения должен соответствовать требованиям

## 7. Доступ к данным

- Доступ предоставляется по принципу минимальной необходимости
- Все доступы должны быть обоснованы
- Доступы должны регулярно пересматриваться

## 8. Передача данных

Разрешается:

- передавать данные только по защищённым каналам
- передавать только необходимый объём информации

Запрещается:

- передавать данные через личные каналы связи
- отправлять данные без проверки получателя

## 9. Удаление данных

- Данные должны удаляться по истечении срока хранения
- Удаление должно быть полным и необратимым
- При необходимости используется анонимизация

## 10. Инциденты

Необходимо немедленно сообщить в IT или ответственному за безопасность:

- утечка данных
- несанкционированный доступ
- потеря устройства с данными
- отправка данных не тому получателю

## 11. Запрещённые действия

Запрещается:

- хранить персональные данные вне утверждённых систем
- копировать данные на личные устройства
- передавать данные третьим лицам без согласования
- использовать данные не по назначению

## 12. Ответственность

Нарушение правил обработки персональных данных может привести к:

- юридической ответственности
- штрафам
- дисциплинарным мерам
