# Работа с электронной почтой

## 1. Назначение документа

Документ определяет правила использования корпоративной электронной почты для обеспечения безопасности и эффективности коммуникаций.

## 2. Область применения

Распространяется на всех сотрудников, использующих корпоративную почту.

## 3. Общие правила

Сотрудник обязан:

- Использовать корпоративную почту только для рабочих задач
- Соблюдать деловой стиль общения
- Проверять получателей перед отправкой
- Не передавать конфиденциальную информацию без необходимости

## 4. Безопасность

- Не открывать подозрительные письма
- Не переходить по неизвестным ссылкам
- Не скачивать вложения от неизвестных отправителей
- Проверять адрес отправителя

## 5. Работа с вложениями

- Открывать только ожидаемые файлы
- Проверять тип и содержимое файлов
- Не запускать исполняемые файлы (.exe, .bat и др.)
- Использовать антивирусную проверку при необходимости

## 6. Конфиденциальность

- Не отправлять чувствительные данные без защиты
- Проверять адрес получателя перед отправкой
- Использовать корпоративные средства передачи файлов вместо вложений при необходимости

## 7. Массовые рассылки

- Использовать рассылки только по необходимости
- Не добавлять пользователей без их ведома
- Использовать скрытые копии (BCC) при внешних рассылках

## 8. Автоматизация

Разрешается:

- использовать фильтры и правила
- настраивать автоответы
- организовывать почту по папкам

## 9. Запрещённые действия

Запрещается:

- использовать личную почту для рабочих задач
- пересылать рабочую переписку на личные адреса
- открывать подозрительные вложения
- переходить по неизвестным ссылкам
- использовать почту для спама или нерабочих целей

## 10. Инциденты

Необходимо сообщить в IT-отдел, если:

- получено подозрительное письмо
- произошёл переход по вредоносной ссылке
- были введены учётные данные на подозрительном сайте
- замечена необычная активность

## 11. Ответственность

Несоблюдение правил может привести к утечке данных, заражению систем и дисциплинарным мерам.
