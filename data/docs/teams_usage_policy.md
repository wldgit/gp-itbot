# Работа с корпоративным мессенджером (Microsoft Teams)

## 1. Назначение документа

Документ определяет правила использования корпоративного мессенджера для рабочих коммуникаций и совместной работы.

## 2. Область применения

Распространяется на всех сотрудников, использующих Microsoft Teams для работы.

## 3. Общие правила

Сотрудник обязан:

- Использовать Teams для рабочих коммуникаций
- Соблюдать деловой стиль общения
- Использовать каналы и чаты по назначению
- Поддерживать порядок в сообщениях и файлах

## 4. Структура коммуникаций

- Каналы — для командной работы и общих обсуждений
- Чаты — для личной или быстрой коммуникации
- Встречи — для обсуждений и синхронизаций

Рекомендуется:

- не дублировать сообщения в разных местах
- использовать каналы вместо личных чатов для рабочих вопросов

## 5. Работа с файлами

- Хранить файлы в соответствующих каналах
- Не дублировать файлы без необходимости
- Использовать единые версии документов
- Не отправлять важные файлы только в личные чаты

## 6. Уведомления

- Настроить уведомления под рабочие задачи
- Не отключать важные уведомления
- Использовать упоминания (@) только при необходимости

## 7. Безопасность

- Не передавать конфиденциальную информацию без необходимости
- Проверять участников чатов и каналов
- Не добавлять внешних пользователей без согласования

## 8. Видеовстречи

- Подключаться вовремя
- Использовать камеру и микрофон по необходимости
- Не распространять ссылки на встречи посторонним

## 9. Запрещённые действия

Запрещается:

- использовать мессенджер для личных целей в рабочее время
- создавать несанкционированные команды и каналы
- передавать доступы и приглашения третьим лицам
- публиковать чувствительные данные без контроля доступа

## 10. Инциденты

Необходимо сообщить в IT-отдел, если:

- обнаружен несанкционированный доступ
- в канал добавлен неизвестный пользователь
- произошла утечка информации

## 11. Ответственность

Несоблюдение правил может привести к утечке данных, нарушению коммуникаций и дисциплинарным мерам.
