# Политика информационной безопасности MedRetail

## Описание

Данный документ содержит основные правила информационной безопасности, которые необходимо соблюдать всем сотрудникам MedRetail.

Соблюдение этих правил помогает защитить корпоративные данные, системы и доступы.

---

## Основные принципы

- каждый сотрудник отвечает за безопасность своего аккаунта  
- доступ к данным должен использоваться только для рабочих задач  
- любые подозрительные действия необходимо сообщать в ИТ-поддержку  

---

## Работа с паролями

Обязательно:

- использовать только свой личный пароль  
- хранить пароль в тайне  
- регулярно менять пароль  

Запрещается:

- передавать пароль другим сотрудникам  
- записывать пароль на видимых местах  
- использовать одинаковые пароли в разных системах  

---

## Работа с почтой

Будьте внимательны при работе с письмами:

- не открывайте письма от неизвестных отправителей  
- не переходите по подозрительным ссылкам  
- не скачивайте вложения, если не уверены в их безопасности  

Признаки подозрительного письма:

- ошибки в тексте  
- срочные требования (например: «срочно оплатить»)  
- неизвестный отправитель  

---

## Использование компьютера

Разрешается:

- использовать компьютер для рабочих задач  
- устанавливать только разрешённое программное обеспечение  

Запрещается:

- устанавливать сторонние программы без согласования  
- отключать антивирус  
- изменять системные настройки без необходимости  

---

## Работа с корпоративными данными

Обязательно:

- хранить данные только в корпоративных системах  
- соблюдать конфиденциальность  

Запрещается:

- копировать данные на личные устройства  
- передавать информацию третьим лицам  
- использовать данные вне рабочих задач  

---

## Использование аккаунтов

- каждый сотрудник должен использовать только свой аккаунт  
- запрещено входить под чужими учетными данными  
- при уходе с рабочего места рекомендуется блокировать компьютер  

---

## Что делать при подозрении на проблему

Немедленно обратитесь в ИТ-поддержку, если:

- вы получили подозрительное письмо  
- заметили необычную активность  
- случайно перешли по подозрительной ссылке  
- подозреваете утечку данных  

---

## Ответственность

Нарушение правил безопасности может привести к:

- блокировке аккаунта  
- ограничению доступа к системам  
- дисциплинарным мерам  

---

## Важно

Соблюдение этих правил — обязанность каждого сотрудника.  
Это помогает защитить данные компании MedRetail и её клиентов.
